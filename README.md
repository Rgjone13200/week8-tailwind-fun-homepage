# week8-tailwind-fun-homepage
Theme type: Futuristic Superhero Luxury Hotel
Hero Haven Hotel is a bold, futuristic hotel website designed for superheroes, featuring sleek rooms, secret headquarters, power-charging stations, and luxury amenities tailored to every hero’s mission. Built with HTML and Tailwind CSS, the site uses a vibrant comic-book-inspired theme with strong colors, glowing effects, and modern layouts to create an exciting and heroic user experience.
